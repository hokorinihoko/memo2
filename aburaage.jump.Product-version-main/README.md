# 油揚げjumpgame
# 作成中　現在修正中にてダウンロードして起動すると動きません
油揚げを天へ導こう
説明
ゲームの簡単な説明：油揚げが上へ進むシンプルゲームで、
リフトに乗りジャンプして高く上りスコアを伸ばすこと目指す
目標：どこまで上に進めるかを競う
操作
space/z/↑/上向きボタン:ジャンプ
r/リスタートボタン:リスタート
m/一時停止ボタン:一時停止
n/再開ボタン:再開
a/←/左向きボタン:左へ移動
d/→/右向きボタン:右へ移動
ルール
キー、ボタン操作で移動やジャンプをしリフトの上に乗り
上へ上へと目指していくゲームです
一定以上上へ行くと下のリフトが消えていきます
1000区切りで報酬サウンド、10000区切りで壮大な報酬サウンドがあります
ゲームオーバー条件は下の画面外へ落ちるとゲームオーバーです
